Baboon

Baboon is a monkey that helps you to detect merge conflicts of a project in a real time.

View the Project on GitHub SeyZ/baboon

What is Baboon ?

After installing Baboon, each developer has a baboon on top of the shoulder. After each file save, Baboon shouts to other baboons the changes of the file. The other Baboons try to apply the changes to their sides and reply if a conflict is detected.

Baboon never changes your code. He just shouts !
